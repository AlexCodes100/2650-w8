This repo contains some boilerplate to get you started in Node with Express

`npm i`

`npm i -g nodemon`

`nodemon server.js`

Create an .env file with the following variables:

GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
SESSION_SECRET=
REDIS_HOST=127.0.0.1
REDIS_PORT=6379
REDIS_PASSWORD=''
MONGODB_URI=
